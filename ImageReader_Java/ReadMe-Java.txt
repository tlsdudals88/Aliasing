You will find
	A program to display and manipulate images. This has been
given as java code to creat an image with a line and then displays it and the processed image on screen (As reference code, both should be 
the same as the original).


Unzip the folder to where you want. To run the code from command line, first compile with:

>> javac imageReader.java

and then, you can run to create an image canvas as:

>> java imageReader


Or, you can install IDE for Java such as eclipse to run a Java project.
